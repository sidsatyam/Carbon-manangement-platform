#  Carbon IoT Device Management Service

##  Overview
The **Carbon IoT Device Management System** is a microservice-based platform for managing IoT devices.  
It supports:
- Device registration, updates, deletions (bulk & single)
- User authentication using JWT
- Parallel user operations
- Dockerized setup for portability
- Automated build, test, and deployment via Jenkins
- RPM packaging for on-prem or OpenStack installation
